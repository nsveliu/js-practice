\#IT Utility Toolkit (Extension)



* Small JavaScript tools created while learning JavaScript fundamentals.



Utilities:

* IP Address Classifier: Validates IPv4 text and detects specific private network ranges.
* Bandwidth Calculator: Handles data rate translations and catches invalid negative metrics.
* Temperature Converter: Executes precise, two-way conversions between Celsius and Fahrenheit scales.
* Uptime Calculator: Translates availability percentages into comprehensive, structured time metrics.



Concepts Practiced - Revisited:

* Variables \& Scope: Managing persistent configurations and tracking mathematical boundaries.
* Functions \& Return Structures: Designing re-usable logic blocks that yield both single string messages and complex, multi-value data properties.
* String Parsing: Navigating and dissecting continuous text blocks character-by-character using specific positioning boundaries (indexOf, substring).
* Template Literals: Formatting dynamic outputs neatly with variables embedded directly into text strings.
* Console Output: Systematically testing functions by logging execution results under various edge-case scenarios.
